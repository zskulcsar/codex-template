# Purpose

Simple template repository to start afresh with:
* codex-cli configured for skills
* Context7 MCP server added 

## Usage

* Create a new repository using this project as template.
* Have your Context7 API key ready and run `setup.sh`
* Run `direnv allow .` to set-up the environment
* Run `codex` to finish the codex setup (login, etc.).
* enjoy!
